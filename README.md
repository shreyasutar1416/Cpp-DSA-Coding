## ▶️ How to Run

Clone the repository

```bash
git clone https://github.com/shreyasutar1416/Cpp-DSA-Coding.git
```

Go to the project directory

```bash
cd Cpp-DSA-Coding
```

Compile

```bash
g++ filename.cpp -o output
```

Run

```bash
./output
```
